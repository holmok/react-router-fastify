# holmok/react-router-fastify

Bare bones react router 7 and fastify

- ssr
- framework mode
- File system routing convention
- no css
- biome
- typescript
- favicons

### Using this...

Clone this repo and run `yarn`

use nvm with `nvm use`

dev = `yarn dev`

build = `yarn build`

run in prod = `yarn start`
